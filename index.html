Game of Life:

Game of Life was my first big project (Made in C++ and wxWidgets). Game of Life was a concept created by the mathmatician John Conway. Rules - Any living cell in the current universe with less than 2 living neighbors dies in the next generation as if by under-population. If a cell meets this criteria in the universe array then make the same cell dead. Any living cell with more than 3 living neighbors will die in the next generation as if by over-population. If so in the universe then kill it. Any living cell with 2 or 3 living neighbors will live on into the next generation. If this is the case in the universe then the same cell lives. Any dead cell with exactly 3 living neighbors will be born into the next generation as if by reproduction. If so in the universe then make that cell alive. My biggest obstacle with this project was implementing the rules they took me far too long to get correct and by the time I had finally gotten them right I had fallen pretty far behind, but I learned something quite valueable which is that no matter how difficult things get you should not quit until the whistle has blown.
